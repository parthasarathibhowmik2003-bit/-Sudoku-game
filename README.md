<h1> link: https://sudoku-game.oneapp.dev/</h1><br># -Sudoku-game
 4 Themes
Black-Blue (Default) - Dark background with blue accents
White-Brown - Cream/beige with brown accents
Black-Green - Dark with green accents
White-Black - Clean white with black accents
✨ Features
3 Difficulty Levels: Easy (35 clues), Medium (45), Hard (55)
Smart Grid: Click any cell to select, highlights row/column and same numbers
Notes Mode: Toggle with button or press 'N' to add pencil marks
Undo: Ctrl+Z or button to reverse moves
Hint System: Reveals correct number (costs 5 points)
Timer & Score: Tracks your solving time and awards points
Mistake Limit: 3 mistakes allowed before game over
Keyboard Support: Use number keys, arrows to navigate, backspace to erase
🎮 How to Play
Select a cell on the 9×9 grid
Click number buttons (1-9) or use keyboard
Toggle "Notes" mode to add small corner marks
Complete the grid so each row, column, and 3×3 box contains 1-9
Avoid mistakes - you only get 3!
